# VocaDay

TOEIC vocabulary learning app.

## Development Environment

- SwiftUI
- SwiftData
- CloudKit private database
- Apple Translation framework
- iOS and macOS Xcode project

## Build

```bash
xcodebuild -project VocaDay.xcodeproj -scheme VocaDay -destination 'generic/platform=iOS' build
xcodebuild -project VocaDay.xcodeproj -scheme VocaDay -destination 'generic/platform=macOS' build
```
