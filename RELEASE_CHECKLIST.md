# VocaDay Release Checklist

## Before TestFlight

- [ ] Test a fresh install and an upgrade from the previous build on an iPhone.
- [ ] Verify CloudKit sync between two devices using the production container.
- [ ] Exercise add, edit, delete, review, grammar notes, backup, and restore.
- [ ] Verify the first-launch spotlight onboarding can be skipped and replayed after reinstall.
- [ ] Confirm the Korean, English, and Japanese product-page metadata and screenshots.

## App Store Connect

- [ ] Upload a signed Release archive and complete TestFlight testing.
- [ ] Provide an iPhone 6.9-inch screenshot showing the Day list and app in use.
- [ ] Complete App Privacy answers: VocaDay stores learning data in the user's private iCloud account and does not track users.
- [ ] Add support URL, privacy-policy URL, age rating, and reviewer notes.
- [ ] Confirm the CloudKit schema is deployed to Production before App Review submission.
